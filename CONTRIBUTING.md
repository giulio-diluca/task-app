# Contributing to Task-App

Thank you for your interest! To keep the code clean and secure, we follow these rules:

1. **No direct commits**: The `main` branch is protected.
2. **Fork & Pull Request**: 
   - Fork the repository.
   - Create a feature branch (`git checkout -b feature/amazing-feature`).
   - Commit your changes.
   - Push to the branch.
   - Open a Pull Request.
3. **Approval**: All PRs require a review and successful status checks before merging.
